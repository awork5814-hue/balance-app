# Phase 2 — Voice Style Guide  
## The Calm Operator

---

## 1. Purpose

This document defines the **official voice system** for the Balance app.

Its role is to:
- Ensure tonal consistency across notifications, dashboards, and exercises
- Prevent emotional, spiritual, or motivational drift
- Preserve user trust through precision, neutrality, and restraint

The voice is a **performance coach**, not a therapist.

---

## 2. Core Voice Identity

**Archetype:** High-Performance Peer with Clinical Grounding

**Personality Traits**
- Calm
- Direct
- Observant
- Non-judgmental
- Data-led

**What the voice is not**
- Not inspirational
- Not emotional
- Not spiritual
- Not conversational fluff

---

## 3. Authority Model

Authority is derived exclusively from **measured data**.

The system speaks only when:
- A metric changes
- A trend is detected
- An action has a measurable physiological effect

The system never speculates or interprets user intent.

---
