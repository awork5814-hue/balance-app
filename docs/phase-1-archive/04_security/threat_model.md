# Threat Model Summary

## Considered Threats
- Device theft
- Shoulder-surfing in public spaces
- Database breach
- Delayed wearable sync confusion

---

## Mitigations
- Biometric Gate + Private Mode
- Field-level encryption
- Partial sync trust logic
- No punitive or alarming UX

---

## Non-Goals
- Clinical diagnosis
- Emergency intervention
- Real-time medical alerts

The app is a coach, not a clinician.
